💡 Definition and Purpose

Analytical Exposition is a type of text that presents arguments to support a writer's opinion. The purpose is to persuade readers that the opinion is valid and logical.

🧱 Generic Structure

  • Thesis: States the writer’s opinion.
  • Arguments: Provides logical reasons to support the opinion.
  • Reiteration: Reinforces and restates the opinion.

🔤 Language Features

  • ✅ Simple present tense
  • ✅ Relational processes (e.g., is, are, has)
  • ✅ Internal conjunctions (e.g., firstly, in addition)
  • ✅ Causal conjunctions (e.g., because, as a result)
  • ✅ Abstract nouns (e.g., freedom, education)
  • ✅ Modal verbs (e.g., should, must)
